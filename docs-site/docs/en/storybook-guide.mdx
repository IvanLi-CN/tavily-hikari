# Storybook Guide

Use this page when you want stable entry points into the UI review surface instead of navigating the
entire Storybook tree manually.

If you are running the public docs-site locally (`bun run dev` or `bun run preview`), start the
Storybook dev server from `web/` first. The redirect links below will target the matching local
Storybook origin.
