# Storybook 导览

这个页面用来提供稳定的 Storybook 深链入口，避免每次都从完整 sidebar 手动翻找。

如果你在本地单独运行 docs-site（`bun run dev` / `bun run preview`），请先在 `web/`
目录启动 Storybook dev server。下面的链接会自动跳到对应的本地 Storybook 地址。
